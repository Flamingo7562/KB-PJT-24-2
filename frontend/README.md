# Frontend

Vue.js 프론트엔드 프로젝트 위치입니다.

## 예정 구조

```text
frontend/
  package.json
  vite.config.js
  eslint.config.js
  .prettierrc
  src/
```

## 실행

Vue 프로젝트 생성 후 아래 명령을 사용합니다.

```sh
npm install
npm run dev
```

## Lint

```sh
npm run lint
npm run lint:fix
```

루트에서는 다음 명령으로 프론트엔드 lint를 호출합니다.

```sh
npm run lint:fe
```
